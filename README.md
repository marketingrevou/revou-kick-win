# RevoU Kick & Win ⚽

A branded penalty-shootout marketing mini-game for RevoU. Pick your country, take three penalty kicks against the keeper, and win a prize.

## Play

Open `index.html` in a browser — it's a self-contained single-page game (HTML/CSS/JS) that loads its art from `assets/` and country flags from `flags/`.

## Structure

- `index.html` — the game (all logic, styling, and markup)
- `assets/` — game art: stadium, ball, keeper sprites, logos, result cards
- `flags/` — country flag images for the selection screen
- `build_scripts/` — Python scripts used to generate/composite game art
